# Proposal: Local contractor quote comparison tool

## Why
Homeowners upload or paste multiple contractor quotes, compare scope gaps, risk, pricing, and negotiation questions before choosing.

## What Changes
- Create a focused full-stack proof of concept for `Local contractor quote comparison tool`.
- Validate whether the idea should be handed to the larger engineering crew.
- Keep the first draft local-first, secret-safe, and deploy-aware.

## Impact
- Adds a draft project under `draft_project/`.
- Adds frontend, backend, database, infra, DevOps, and deployment documentation.
- Defines implementation requirements in `specs/`.
